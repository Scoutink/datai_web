    1. Find your documents in All Documents or Assigned Documents.
    2. Upload using Add Document or Bulk Upload for multiple files.
    3. Control access via Share Document or a Shareable Link.
    4. Stay on track with Reminders and Notifications.

    CHAPTER 03

    Document Library

    Where users browse, filter, and monitor the full lifecycle of every document.

    All Documents

    The All Documents screen centralizes search, filters, and action controls for every file in the platform.

    • Search by keyword and filter by category, status, owner, and date ranges.
    • Use sorting controls for title, date, and relevance.
    You can select one, multiple, or all documents and use the bulk Share option to distribute them to users or groups in a single action.

    Assigned Documents

    Assigned Documents focuses on files shared directly with you and includes task-centric controls such as status updates and reminders.

    • Status dropdown and My Reminders panel are specific to assigned views.
    • Each entry shows name, status, category, creation date, expiration date, and created by.

    Archived Documents

    Archived Documents lists files kept for historical retention and review without showing them in active workflows.

    Expired Documents

    Expired Documents lists files that have crossed configured validity windows and may require renewal or replacement.

    Expired documents are automatically flagged by the system. This prevents outdated versions from being used in active processes.

    Search

    Find any document — by name, by tag, or by what's written inside it.

    Standard Search

    Standard Search filters records by common metadata fields.

    • Name
    • Meta tags
    • Category
    • Storage
    • Creation date

    Deep Search

    Deep Search scans document content to surface semantically related matches and high-value textual hits.

    • Top 10 results
    • Case-insensitive
    • Stop words excluded
    • Word stemming enabled
    • Supported across major office and PDF file types
    Searching "run" will also return results for "running" and "runs". Common words like "and", "the", and "is" are automatically excluded to improve precision.
